# Changelog

Initial WinUI 3 release.

- Rebuilt XDB-database as a C# WinUI 3 desktop app.
- Added dark dashboard UI inspired by the provided mobile mockups.
- Added service cards for Apache and MySQL/MariaDB.
- Added active service summary card.
- Added Logs, PHP Switcher, and Tools views.
- Added Administrator-aware Apache service start/stop handling.
- Added local MSIX signing/install scripts for development release builds.
