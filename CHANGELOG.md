# Changelog

## v1.0.0 - 2026-05-03

Initial WinUI 3 release.

- Rebuilt XDB-database as a C# WinUI 3 desktop app.
- Added dark dashboard UI inspired by the provided mobile mockups.
- Added service cards for Apache and MySQL/MariaDB.
- Added active service summary card.
- Added Logs, PHP Switcher, and Tools views.
- Added Administrator-aware Apache service start/stop handling.
- Added local MSIX signing/install scripts for development release builds.

### Fixed after release review

- Safer MySQL/MariaDB stop behavior that avoids force-killing database processes after failed graceful shutdown.
- Safer process detection so unrelated Apache/MySQL processes are not treated as XAMPP processes.
- Apache service actions now check elevated command exit codes and stop timeout.
- Release zip now includes Windows App Runtime dependency packages generated by MSIX packaging.
- Installer now trusts the development certificate for the current user instead of the machine Root store.
- Removed unused `systemAIModels` capability from the package manifest.
- Improved dashboard and tools responsiveness for narrower desktop windows.
- Added Settings page for XAMPP root, Apache/MySQL service names, and preferred browser.
- Added app activity logging under the current user's LocalAppData folder.
- Added startup validation for required XAMPP folders and key executable/config files.
- Added phpMyAdmin access repair helper with automatic backup and Apache syntax validation.
- Release packaging now emits an `.msixbundle` and includes a friendlier one-click install batch file.
- Added service transition states with progress indicators and persistent error state.
- Added port conflict detection with PID and process names for ports 80, 443, and 3306.
- Added dashboard health summary for Apache, MySQL, phpMyAdmin, and PHP version mismatch.
- Expanded Tools with quick access to `php.ini`, `my.ini`, `httpd.conf`, `httpd-vhosts.conf`, `error.log`, and Windows Services.
- Added PHP Switcher preview, stricter PHP file validation, confirmation, and rollback to the last backup.
- Added portable stack foundation with `config`, `stack`, `www`, `data`, `apps`, and `scripts` folders.
- Added dynamic Apache/MariaDB config rendering with runtime path placeholders.
- Added direct process control for Apache and MariaDB without Windows Services.
- Added MariaDB data initialization when `mariadb-install-db.exe` is available.
- Added port fallback support for Apache 8080 and MariaDB 3307 when default ports are busy.
- Added portable shell helper and GitHub Actions Windows release workflow.
