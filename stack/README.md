# XDB Portable Stack

Place portable server binaries here:

- `stack/apache` - Apache Lounge extracted folder, with `bin/httpd.exe`.
- `stack/mariadb` - MariaDB portable extracted folder, with `bin/mysqld.exe`.
- `stack/php` or `stack/php8.2` - PHP for Windows extracted folder, with `php.exe` and `php8apache2_4.dll`.

XDB-database renders runtime configuration from `config/*.template.*` each time a service starts.
